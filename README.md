# القرآن الكريم — تطبيق سطح المكتب (WPF / .NET 9)

تطبيق سطح مكتب لعرض القرآن الكريم وتصفّحه، مع بحث متقدّم، وإعراب، وتفسير، وتلاوة صوتية.

> تطوير: **شركة الرائد للحلول البرمجية**

## المزايا

- **عرض المصحف** بالرسم العثماني، بتبرير متّصل مثل المصحف الحقيقي، صفحة واحدة أو صفحتين متقابلتين.
- **بحث احترافي** في كل المصحف: نصّي وبجذر الكلمة، يتجاهل التشكيل ويوحّد الحروف (ة/ه، الألفات، الياء)، ويطابق الرسمين الإملائي والعثماني.
- **الإعراب** لكل كلمة (نوعها وحالتها وعلامة إعرابها) من Quranic Arabic Corpus.
- **التفسير الميسَّر** لكل آية (مجمع الملك فهد) — يعمل أوفلاين.
- **التلاوة الصوتية (ترتيل)** لأربعة قُرّاء، مع تحميل مسبق وتشغيل أوفلاين بعده، وتمييز الآية الجارية.
- **إحصائيات البحث**، نسخ الآيات (مع/بدون تشكيل ومعلومات)، أرقام الصفحات والأجزاء والأحزاب والسجدات.
- واجهة **تبويبات** (مثل المتصفّح) مع صفحة إعدادات كاملة وإدارة تحميل الصوت.
- **نظام تحديث تلقائي** (AutoUpdater.NET) ونشر **self-contained** لا يحتاج تنصيب .NET.

## البنية

| المشروع | الوصف |
|---------|-------|
| `QuranAlKareem.App` | واجهة WPF (MVVM) |
| `QuranAlKareem.Core` | النماذج والخدمات ومنطق تطبيع النص العربي |
| `QuranAlKareem.Data` | قاعدة SQLite والاستيراد والوصول للبيانات |

## مصادر البيانات

- نص القرآن (عثماني/مبسّط) وأرقام الصفحات/الأجزاء: AlQuran Cloud.
- التحليل الصرفي (الجذور/الإعراب): Quranic Arabic Corpus.
- التفسير: التفسير الميسَّر (`ar.muyassar`).
- الصوت: everyayah.com (بثّ ثم تخزين محلي).

## البناء والتشغيل

```bash
dotnet build
dotnet run --project QuranAlKareem.App
```

عند أول تشغيل تُبنى قاعدة البيانات `quran.db` محلياً من ملفات `QuranAlKareem.App/Data/`.

## الإصدار

```powershell
powershell -ExecutionPolicy Bypass -File build-release.ps1 -Version 1.0.1
```

ينتج نسخة self-contained مضغوطة في `release/` وملف `deploy/VersionInfo.xml` للتحديث.
